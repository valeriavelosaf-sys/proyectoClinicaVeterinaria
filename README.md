Sistema Clínica Veterinaria Vida Animal


Este proyecto permite gestionar la información de las mascotas atendidas en la clínica veterinaria **Vida Animal**, así como los datos de sus dueños y el cálculo del costo de consulta.

 Funcionalidades
- Registro de dueños
- Registro de mascotas
- Asociación dueño–mascota
- Cálculo del costo de consulta veterinaria
- Visualización de información


 Cálculo de consulta
El costo se calcula con un valor base y recargos dependiendo de:
- Especie del animal
- Edad

 Tecnologías usadas
- IntelliJ IDEA
- Git & GitHub
